# 🚀 Crypto Trading Tools

Herramientas de trading para encontrar memecoins antes de que exploten.

## 📱 Aplicaciones

### 1. Token Profile Scanner
- Escanea tokens de DexScreener Token Profile
- Simulador de inversión ($5 por token)
- Venta automática en pérdida/ganancia
- Trailing Stop integrado
- Selección de redes blockchain

**Archivo:** `token-profile-scanner/index.html`

### 2. Crypto Hunter Pro
- Scanner en tiempo real de múltiples APIs
- Alertas de volumen y precio
- Notificaciones Push
- Watchlist personal
- Calculadora de ganancias
- Diseño móvil optimizado

**Archivo:** `crypto-hunter-pro/index.html`

## 🚀 Cómo usar

### Opción 1: Local
1. Clona el repositorio
2. Abre el archivo HTML en tu navegador
3. ¡Listo!

```bash
# Para correr un servidor local
cd token-profile-scanner
python -m http.server 8080
```

### Opción 2: Subir a Netlify/Vercel
1. Arrastra la carpeta a Netlify Drop
2. ¡Listo! URL pública gratis

## 🔧 Configuración

### Token Profile Scanner
- Redes: Solana, Ethereum, BSC (seleccionables)
- Inversión por token: $5 (configurable)
- % Venta: configurable

### Crypto Hunter Pro
- Refresh: 15 segundos (configurable)
- Alertas: % mínimo configurable
- Sonido: opcional

## 📊 Características

- ✅ Datos de DexScreener API
